# SUDHA (सुधा) - Healthcare Safety Platform

## About SUDHA

**SUDHA** (meaning "pure nectar" in Sanskrit) is an Indian healthcare safety platform built with Ayurvedic wisdom and modern technology. 

The platform provides:
- Medicine authenticity verification
- Symptom awareness guidance
- Food-drug interaction alerts
- Ayurvedic and Vedic health insights
- Emergency SOS support

**Medical Disclaimer**: This platform is for informational purposes only. It does not diagnose, prescribe, or replace professional medical advice.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm i

# Step 4: Set up Firebase configuration
# Copy .env.example to .env and add your Firebase credentials

# Step 5: Start the development server
npm run dev
```

The app will be available at `http://localhost:8080`
